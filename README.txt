This repository contains scripts and necessary files to analyse ITS2 amplicons using DADA2.

DADA2 is run through a conda environment (downloadable QIIME2-2018.11 release environment). 

Sources for scripts: 
https://benjjneb.github.io/dada2/ITS_workflow.html
https://astrobiomike.github.io/amplicon/dada2_workflow_ex
